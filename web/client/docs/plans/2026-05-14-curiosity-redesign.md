# Curiosity Trail 重新设计方案

## 1. 设计概述

### 1.1 目标
本设计方案旨在全面提升 Curiosity Trail 科学探索记录工具的视觉风格、用户体验和功能完整性。

### 1.2 设计方向
- **视觉风格**: 学术专业风格，类似学术期刊设计语言
- **用户体验**: 简化操作流程，增强交互反馈
- **功能增强**: 添加智能搜索、数据可视化、模板系统

### 1.3 技术栈
- React 18 + Vite
- Ant Design 5.x
- Recharts（数据可视化）
- Lucide React（图标）

## 2. 视觉风格设计

### 2.1 色彩方案

| 用途 | 浅色模式 | 深色模式 | 说明 |
|------|---------|---------|------|
| 主色 | #1e3a5f | #3b82f6 | 深靛蓝，沉稳学术感 |
| 辅助色 | #6b7280 | #9ca3af | 暖灰，正文使用 |
| 强调色 | #d97706 | #f59e0b | 琥珀金，重点标注 |
| 成功 | #10b981 | #10b981 | 绿色 |
| 警告 | #f59e0b | #f59e0b | 橙色 |
| 错误 | #ef4444 | #ef4444 | 红色 |
| 背景 | #fefefe | #111827 | 页面背景 |
| 卡片 | #ffffff | #1f2937 | 卡片背景 |
| 边框 | #e5e7eb | #374151 | 分隔线 |

### 2.2 字体方案

| 用途 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 标题 | Noto Serif SC | 24-32px | 600-700 |
| 副标题 | Noto Sans SC | 16-18px | 500 |
| 正文 | Noto Sans SC | 14px | 400 |
| 辅助文字 | Noto Sans SC | 12px | 400 |
| 代码 | JetBrains Mono | 13px | 400 |

### 2.3 布局规范

- **侧边栏宽度**: 260px
- **顶部栏高度**: 64px
- **卡片圆角**: 12px
- **间距基准**: 8px（所有间距为 8px 的倍数）
- **内容区边距**: 24px（桌面）/ 16px（移动端）

## 3. 用户体验改进

### 3.1 报告编辑体验

| 功能 | 描述 | 优先级 |
|------|------|--------|
| Markdown 编辑 | 支持 Markdown 语法的富文本编辑 | 高 |
| 实时预览 | 左侧编辑，右侧实时预览 | 高 |
| 自动保存 | 输入停止 2 秒后自动保存，显示状态 | 高 |
| 章节导航 | 显示报告大纲，可快速跳转 | 中 |
| 快捷键支持 | Ctrl+S 保存、Ctrl+B 加粗等 | 中 |
| 字数统计 | 底部显示字数和阅读时间估算 | 低 |

### 3.2 文献管理效率

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 拖拽上传 | 支持拖拽文件到页面直接上传 | 高 |
| 上传进度 | 显示上传进度条和队列状态 | 高 |
| 智能筛选 | 按名称、日期、分类筛选排序 | 高 |
| 快速操作 | 悬停显示快捷按钮，双击打开 | 中 |
| 批量操作 | 支持多选批量删除/移动 | 中 |

### 3.3 报告与文献关联

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 拖拽关联 | 可拖拽文献到报告建立关联 | 高 |
| 关联面板 | 编辑报告时底部显示已关联文献 | 高 |
| 双向导航 | 从文献可跳转到关联报告 | 中 |
| 批量关联 | 支持多选文献批量关联到报告 | 中 |

## 4. 功能增强

### 4.1 智能搜索

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 全局搜索 | 搜索报告标题和内容 | 高 |

| 类型筛选 | 只搜索报告/只搜索文献 | 中 |
| 搜索建议 | 实时显示搜索建议和历史 | 中 |
| 搜索历史 | 保存最近 10 条搜索记录 | 低 |

### 4.2 数据可视化

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 统计仪表板 | 报告总数、文献总数、分类分布 | 高 |
| 趋势图表 | 每月新增报告趋势柱状图 | 中 |
| 分布图表 | 文献分类分布饼图 | 中 |
| 关联统计 | 报告与文献关联分析 | 低 |

### 4.3 模板系统

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 模板导航 | 显示科学探究模板大纲 | 高 |
| 章节跳转 | 点击快速跳转到指定章节 | 高 |
| 填写提示 | 显示当前章节的填写建议 | 中 |
| 自定义模板 | 支持创建和管理自定义模板 | 低 |

## 5. 页面结构

```
src/
├── App.jsx                    # 主应用组件
├── main.jsx                   # 入口文件
├── components/
│   ├── Reports.jsx            # 报告列表页
│   ├── Documents.jsx          # 文献列表页
│   ├── ReportEditor.jsx       # 报告编辑器
│   ├── Settings.jsx           # 设置页（含统计仪表板）
│   ├── Sidebar.jsx            # 侧边栏组件
│   ├── Header.jsx             # 顶部栏组件
│   ├── SearchBar.jsx          # 搜索组件
│   └── StatDashboard.jsx      # 统计仪表板
├── hooks/
│   ├── useSearch.js           # 搜索 Hook
│   └── useTheme.js            # 主题 Hook
└── utils/
    └── helpers.js             # 工具函数
```
